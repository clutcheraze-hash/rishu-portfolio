# Kai Tanaka — Premium Portfolio

A premium personal portfolio website built as a cinematic single-page experience, inspired by modern Japanese digital design studios, Tokyo luxury aesthetics, and award-winning interactive experiences.

## Overview

This is not a traditional portfolio. It's a **digital experience** designed to communicate innovation, precision, discipline, and futuristic creativity — the kind of presence expected from a founder operating at the intersection of esports, technology, and creator culture.

## Key Technologies

| Technology | Purpose |
|---|---|
| [TanStack Start](https://tanstack.com/start) | Full-stack React SSR framework |
| [Tailwind CSS v4](https://tailwindcss.com) | Utility-first styling |
| [Lenis](https://lenis.studiofreight.com) | Cinematic smooth scrolling |
| [Framer Motion](https://www.framer.com/motion/) | Physics-based animations |
| [GSAP](https://greensock.com/gsap/) | High-end scroll-triggered animations |
| [Netlify](https://netlify.com) | Hosting, CDN, serverless functions |

## Design System

- **Palette**: Deep black `#050505`, charcoal, off-white `#e8e4dc`, metallic silver, crimson `#c8102e`
- **Typography**: Cormorant Garamond (luxury serif display) + Space Mono (technical mono) + Noto Serif JP (authentic Japanese)
- **Motion**: CSS scroll-reveal, Lenis smooth scrolling, particle system, custom magnetic cursor

## Running Locally

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000).

## Sections

1. **Hero** — Full-screen immersive landing with animated name reveal and floating particles
2. **About** — Story-driven presentation with portrait and statistics
3. **Experience** — Entrepreneurship journey across esports, talent management, and startups
4. **Skills** — Interactive skill visualization with animated progress bars
5. **Projects** — Case-study style venture cards
6. **Achievements** — Animated counter statistics
7. **Contact** — Minimalist Japanese-inspired contact form
